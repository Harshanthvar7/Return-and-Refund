# Secure Return & Refund Management System

A role-based, secure prototype for handling product return, refund, and customer support queries.

## Roles
- User
- Customer Support
- Admin

## Features
- Secure authentication
- Role-based access control
- Return requests with evidence
- Customer support query system
- Admin approval workflow
- User-only refund notifications
